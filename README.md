# IOT-DOOR-LOCK
This is a door lock that is controlled using an Arduino Nano 33 IOT and the Arduino cloud. The system uses a relay module, 12v electromagnetic lock, buzzer module and a 12v power supply. The wiring diagram is in the comments of the code. The lock is controlled from the cloud dashboard using different widgets. To allow multiple users you can share the dashboard to the poeple you want to. you can also remove acces to the dashboard.


How To Run:

1. Configure board to cloud
2. Upload iot_door_lock.ino to board from cloud
3. Set up dashboard/Variables on cloud
4. System is now ready
